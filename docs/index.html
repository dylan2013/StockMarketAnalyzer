如何取得已實現損益 .CSV

使用「樹精靈」App，點擊「帳務」→「已實現」→「篩選」按鈕。

點擊「選擇區間」→「指定（起訖日）」。查詢區間最長為 5 年；若在上午 13:30 前執行，當天僅能查詢 1 年。

查詢完成後，點擊「匯出」將結果匯出。

匯出方式可選擇「Line 分享」，分享至特定 Line 帳號，再從 Line 下載『.csv檔案』至電腦。

以下為 Excel 內容範例，確認欄位正確後即可上傳。

如何取得配息紀錄 .MHT

使用國泰證券 App，進入「我的」→「配股配息」。

點擊「區間篩選」按鈕。

選擇查詢區間，建議以 3 年為單位分批操作（最早僅能查詢至 2018 年）。

點擊右上角「使用瀏覽器開啟」，以便保存畫面內容『.mht』。

選擇「使用 Chrome 瀏覽器」開啟。

在 Chrome 中點擊右上角選單，再點擊「下載」按鈕，將檔案儲存至 Download 資料夾『.mht』。
